It is not affiliated with, endorsed by, or sponsored by the original Quill project or its contributors.

The goal is to...

- keep quill maintained and up to date,
- address security issues,
- work on known bugs and quirks

We do not focus on new features and major architectural redesign. If you are interested in such, feel free to check out [notectl](https://github.com/samyssmile/notectl).

## How to join and contribute?

We have a [Governance repo](https://github.com/quillcommunity/governance) which we use to plan and discuss topics, such as...

- which repo(s) to fork and how to continue them
- how we want to work together
- community rules for joining, contributing, communication etc.
- safety measures and security related topics in this community
- rules for AI usage
- licensing questions

Technical discussion and contribution is done instead in the [Quill fork](https://github.com/quillcommunity/quill) repo, including...

- copied issues / PRs from the upstream repo (the original Quill repo)
- new issues/discussions related to Quill functionality
- safety measures and security topics related to Quill itself (including CVEs of Quill and its dependencies)
